# xsvCommandCenter - Project Manifest

## 1. Prime Directives
1. **Hybrid Portability:** Works on USB, SSD, or Cloud.
2. **Native Integration:** Runs in native terminal.
3. **Cross-Platform:** Uses pathlib and os.name checks.

## 2. Module Registry
| Module | Purpose |
| :--- | :--- |
| **Core** | Auth, Config |
| **Vault** | Encrypted storage |
| **Host** | OS Interaction |
